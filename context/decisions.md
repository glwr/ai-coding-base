# Decisions Log

> Architecture and design decisions. ADR-style. Append-only — never delete entries, only supersede them.

## How to Read
- Entries are in reverse chronological order (newest first)
- Superseded decisions are marked with `**Status: Superseded by [TITLE]**`
- Search by feature ID: `PROJ-X`

## Entry Format
```
### [Short Title]
- **Date:** YYYY-MM-DD
- **Feature:** PROJ-X (or "General")
- **Skill:** /skillname
- **Decision:** What was decided
- **Rationale:** Why this was chosen
```
